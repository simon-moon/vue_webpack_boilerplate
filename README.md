# new_schweser

> Schweser as Vue.js App

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# view production build at localhost:8080
cd dist
http-server ./

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
